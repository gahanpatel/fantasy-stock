## 📌 Project Description

Fantasy Stock League is a full-stack web application where users simulate stock trading in a competitive league format. Users create virtual portfolios, execute mock trades using real-time or periodically updated stock prices, and compete on a live leaderboard based on portfolio performance.

The backend handles price updates, portfolio valuation, and ranking logic, while the frontend provides an interactive dashboard for tracking trades and performance. 

The application will be deployed on Vercel for seamless hosting, continuous integration, and automatic updates.
